About the Map

This map pulls live data from Esri, the WAQI, and Open-Mateo API to generate an up-to-date map of wildfires, air quality, and wind patterns across the United States of America. These layers can be viewed independently or all at once. There are also three basemaps to choose from: streets, satellite imagery, and terrain. Clicking on wildfire polygons displays a pop-up with the fire's name, size, location, and a link to more information. The air quality layer shows the measurements of Air Quality Index stations nationwide. And the wind layer shows the direction and speed of winds from sea to shining sea! This map is meant to be accessible to all people, though Americans will find it most useful.

The web map was built with Leaflet using base layers from OpenStreetMap and Esri. Most datasets were already compatible with web mapping so little was done to filter or sort the data. Wildfire data was accessed directly from an ArcGIS REST API and rendered as GeoJSON, U.S. boundary data was also retrieved as GeoJSON and used to create the mask. Wind data was gathered by sampling a grid of latitude and longitude points and querying the Open-Meteo API, then the symbols were customized to an instantly readable state. The number of wind requests was limited to improve performance. All data was used in WGS84, which is the standard web mapping coordinate system and default for Leaflet. Final data delivery to the map happened using live APIs and GeoJSON services, as well as tiled map layers for air quality, allowing good visualization without downloading or storing local datasets.
